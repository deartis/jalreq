# ⚡ JAL REQ

> **JAL REQ** é um cliente de testes de API (API Client) leve, rápido, moderno e 100% focado em privacidade e produtividade. Desenvolvido em Flutter, ele foi feito para desenvolvedores que precisam testar suas requisições HTTP sem a lentidão ou as distrações das ferramentas comerciais tradicionais.

---

## ✨ Funcionalidades principais

- 🚀 **Requisições HTTP Rápidas**: Suporte completo a `GET`, `POST`, `PUT`, `PATCH` e `DELETE`.
- 📁 **Organização por Coleções**: Agrupe suas requisições relacionadas e salve-as para acesso rápido.
- 🕒 **Histórico de Requisições**: Histórico local de testes anteriores para fácil reexecução.
- 🔐 **Headers, Body & Auth**:
  - Configuração flexível de cabeçalhos.
  - Suporte a múltiplos tipos de corpo (`raw` JSON, `form-data`, `x-www-form-urlencoded`).
  - Abas dedicadas a autenticação rápida.
- 🌐 **Variáveis de Ambiente**: Defina ambientes (ex: Desenvolvimento, Homologação, Produção) e gerencie variáveis dinâmicas nas suas URLs e cabeçalhos.
- 📋 **Integração com cURL**: Copie qualquer requisição configurada como comando `cURL` com apenas um clique.
- 💾 **Exportação & Importação**: Salve e carregue suas coleções e configurações em arquivos JSON.
- 🎨 **Interface Premium & Dark Mode**: Visual minimalista e moderno em tons escuros para evitar fadiga ocular.

## 🚀 Como começar

### Pré-requisitos
Certifique-se de ter o **Flutter SDK** instalado em sua máquina. Para verificar, execute:
```bash
flutter doctor
```

### Instalação e execução
1. **Clone o repositório:**
   ```bash
   git clone https://github.com/deartis/jalreq.git
   cd jalreq
   ```

2. **Obtenha as dependências do Flutter:**
   ```bash
   flutter pub get
   ```

3. **Execute o aplicativo:**
   ```bash
   flutter run
   ```
